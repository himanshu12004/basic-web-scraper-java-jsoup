# Basic Web Scraper for Headlines

## Description
This project is a basic web scraper developed using Java and the Jsoup library.  
It fetches the top 10 news headlines from a news website and displays them in a clean HTML format.

The project demonstrates core concepts of web scraping such as sending browser-like requests, parsing HTML documents, and extracting meaningful data.

---

## Features
- Fetches top 10 latest news headlines
- Uses Jsoup for HTML parsing
- Mimics browser requests using headers
- Generates an HTML file for easy viewing
- Simple and beginner-friendly implementation

## How It Works
1. The Java program connects to a news website using Jsoup.
2. It parses the HTML document.
3. Headline elements are extracted using CSS selectors.
4. Top 10 headlines are written into an HTML file (`news.html`).

## How to Run
1. Clone the repository:
   git clone https://github.com/himanshu12004/basic-web-scraper-java-jsoup.git
2. Open the project in IntelliJ IDEA or any Java IDE.
3. Run the `Main.java` file.
4. Open `news.html` in a browser to view the headlines.
